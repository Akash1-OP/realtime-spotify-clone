# 🎵 Realtime Spotify Clone

A full-stack Spotify-inspired music streaming platform with real-time social features, built using modern web technologies. Users can stream music, chat with friends, see live listening activity, and interact in real time.

## 🌐 Live Demo

https://spotify-advance-clone-f5j9.onrender.com



![Realtime Spotify Clone](./frontend/public/Screenshot%202026-06-20%20000130.png)

---

## ✨ Features

### 🎧 Music Streaming
- Play, pause, skip, and control volume
- Album and song management
- Responsive music player
- Real-time listening experience

### 💬 Real-Time Social Features
- Live chat between users
- Online/offline presence tracking
- Real-time activity updates
- Instant message delivery using Socket.IO

### 🔐 Authentication & Authorization
- Secure user authentication
- Protected routes
- User profile management

### 📊 Admin Dashboard
- Manage songs and albums
- Upload music and cover images
- View platform analytics
- User activity monitoring

### ☁️ Media Management
- Cloud-based image and audio storage
- Optimized media delivery
- Secure file uploads
